aperture-vman

Volunteer Management System created for PAST&E by CS Capstone Team Aperture. This project’s goal is to create a web application to help PAST&E’s staff keep track of their volunteers’ schedules, and to help facilitate communication. This was created using a Drupal 7 installation as well as several additional modules and custom code.

To view the user guide, go here.

Vman can:

  • Potential volunteers can apply for new accounts
  • Volunteer accounts can be managed by the admin
  • Volunteers and staff can be grouped into which age groups they will be working with
  • Admin and staff can send messages to each other or to volunteers to keep them updated on current news and information
  • Volunteers can enter theirs schedule into the calendar so that the admin and staff can easily keep track of who is working when and with which group
  • Admin and staff can update calendar entries to update or remove a volunteer’s time as needed
  • Admin can run reports on the volunteers time to see how many people have volunteered how many hours
